Week 6

JQuery

Your learning challenge this week will be about jquery. For the team learning component, your objective is to re-implement the Yelp site, using $.get() to dynamically pull data in and using DOM insertion (i.e., .html()) to to display the data. For the individual learning component, your objective is to re-implement the Github site you worked on last week, but instead of grabbing data from a MongoDB server, you will be pulling in data using the Github API.

What you are building right now is a simple version of the so-called single-age app. Think about how this differs from the static site generator (Wintersmith) and the server-side app (Express) in terms of pros and cons.
